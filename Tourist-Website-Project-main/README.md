# Tourist Website Project

A tourism-themed website developed as a collaborative university project during the 1st semester.

## Features

- Home page
- Tourist destinations
- Gallery section
- Packages page
- Contact page
- FAQs section
- Responsive design elements

## Technologies Used

- HTML5
- CSS3

## Project Type

Academic Group Project

## Contributors

This project was developed collaboratively by a team of 3 students as part of a university coursework project.

## Purpose

The purpose of this project was to practice:

- Frontend web development
- Website structuring
- UI design basics
- Team collaboration
- GitHub project hosting
