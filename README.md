# Tourist Website Project 

A tourism-themed frontend website developed as a collaborative university project during the 1st semester.

## 📌 Overview

This project was created to showcase popular tourist destinations and travel packages through a visually appealing and user-friendly website interface.

The website includes multiple pages such as destinations, gallery, packages, FAQs, contact section, and more.

---

##  Features

- Responsive homepage
- Tourist destinations showcase
- Travel packages section
- Image gallery
- FAQ page
- Contact page
- Clean frontend UI
- Multi-page website structure

---

##  Technologies Used

- HTML5
- CSS3
